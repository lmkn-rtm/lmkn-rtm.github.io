Overview

Atum is a meta-library combining multiple specialized libraries under a unified interface for wide range of utilities for windows desktop application development. All external libraries were compiled (or extracted from archive) to object files and included in libatum.a to specify less flags during linking. No changes to external libraries were made (unless specified explicitly) and all licenses can be found in respective folders at "./src/external/".

Core Modules

atum.h

Header that includes headers of all modules for you

config.h

Contains macros for configuring lib (rebuild the lib if you want to apply changes)

atim_utils.h

Contains utility functions, logging, basic types and macros

atum_data.h

Contains data structures, algorithms, threading and some math

atum_io.h

Contains functions for working with directories, some system data and json files

atum_core.h

Contains window utilities; core module for graphical applications

Installation

  1. Download

    Get the latest release from GitHub Releases

  2. Extract

    Unzip to your preferred location.

  3. Build

    If you want to recompile the library, open console in src directory and run make.

  4. Link

    Add these flags to you Makefile (or console, its just a reference):

    
                    SET ATUM_PATH="where you put libatum folder"
                    SET IPATH=-I$(ATUM_PATH)/src
                    SET LPATH=-L$(ATUM_PATH)
                    SET LFLAGS=-latum -lopengl32 -lgdi32

    -I specifies location of headers (in src there are both .h and .c files as well as Makefile to build it all.

    -L specifies where libatum.a is - it's in libatum folder with README and LICENSE.txt

    -l flags are used for lib linking, -lopengl32 and -lgdi32 are for stuff glfw uses, i cannot remove them by adding object files to libatum.a because these are dlls...
