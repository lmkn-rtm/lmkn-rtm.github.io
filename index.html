Welcome to Aspects of Mire Wiki

About

Aspects of Mire is a sandbox voxel game written with C and Raylib. It add a whole new world with lore, survival, mechanics, story and music. Dive into the hostile but beatiful world and start your journey in the lands of mire.

The game is currently under development so make sure to check new versions and veatures on this wiki. If you want to report bugs or suggest a feature - raise an issue on GitHub repository of the project (here).

WARNING! - Some versions may be incompatible with previous releases because of changes in core structure. Always read CHANGELOG to know if it is possible to port your older worlds to the newer version. Sometimes convertion apps are released for world update, but it't not a consistent occasion (may never happen at all) and generally not a good idea.
